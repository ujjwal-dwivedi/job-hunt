# Job Portal Web Application

A basic job portal web application that connects recruiters with job seekers.

## Live Link
[job-hunt-eta-puce.vercel.app](https://job-hunt-eta-puce.vercel.app)

## Features

### Recruiters
- Register and manage company profiles
- Post new job openings
- Edit or delete job listings
- View applications from candidates

### Job Seekers
- Create and manage user profiles
- Search and filter job listings
- Apply for jobs online
- Track application status

## Technologies Used
- Frontend: React, TailwindCSS, Javascript
- Backend: Node.js, Express.js, JWT
- Database: MongoDB
